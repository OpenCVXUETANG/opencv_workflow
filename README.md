# OpenCV Experiment Master Toolkit Software
The primary application scenario is in the field of machine vision. The OpenCV low-code development workflow engine SDK supports C++, C#, and Python, enabling 80% of OpenCV development work with zero code and increasing OpenCV development efficiency by ten times。support visual design the opencv operator workflow, save and export, reload and re-use。already support to find circles/corners/lines/contours/points/blobs, image preprocess, image enhancement(contrast adjustment,brightness adjustment, histogram equalization, denoising, sharpening), image morphological processing, template match(NCC, Edge), Caliper Measurement(line\edge\circle). the whole platform include opencv workflow SDK, OpenCV operator Design toolkit software, sample codes and C#/C++ support libs, all of these tools only depend on opencv,the latest opencv version is opencv4.10
